# ESTRATERIK — Dashboard de Consultores

Dashboard online de acompanhamento dos consultores da Oikos Assessoria.

## Tecnologias
- Frontend: HTML + CSS + JavaScript (arquivo único)
- Banco de dados: Supabase (PostgreSQL)
- Hospedagem: GitHub Pages

## Como configurar o banco de dados
1. Acesse o projeto no Supabase
2. Vá em **SQL Editor**
3. Execute o arquivo `setup_banco.sql`

## Link do dashboard
Disponível via GitHub Pages após ativar nas configurações do repositório.
